## 📝 Descripción del Proyecto

**PostWatch** es una plataforma de comercio e-commerce orientada a la venta de pósters, afiches y arte impreso de películas y series icónicas de la cultura pop y producciones actuales. 

El objetivo principal del proyecto es ofrecer una experiencia fluida e intuitiva para los fanáticos del cine que buscan decorar sus espacios, permitiéndoles explorar un catálogo interactivo y gestionar sus productos preferidos.

### ✨ Características Principales
*   **Catálogo de Productos:** Visualización de pósters con precios actualizados y portadas.
*   **Sección de Detalle:** Acceso a información específica de cada póster.
*   **Carrito de Compras:** Sistema dinámico para añadir productos y gestionar las cantidades para la compra final.
*   **Lista de Favoritos (Wishlist):** Opción interactiva (icono de corazón) para que los usuarios puedan guardar los pósters que desean comprar más adelante.
*   **Navegación:** Menú superior con acceso rápido a secciones como "Sobre Nosotros", carrito y elementos guardados (Favoritos).

---

## 🛠️ Lenguajes Utilizados

El desarrollo del lado del cliente (Frontend) se construyó utilizando:
*   **HTML5 & CSS3:** uso de paleta de colores blanco, negro y tonos rojos para simular la estética de una sala de cine.
*   **JavaScript (ES6+):** Para la interactividad de los botones de favoritos y carrito, y para el manejo de sus desplegables.
